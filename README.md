# Dog Butterfly Chase

A small pixel-style browser game where a cheerful dog catches butterflies in a bright meadow.

## Play

Open `index.html` in a browser.

Press **Start**, then click butterflies before the 30-second timer reaches zero. The dog will run toward the selected butterfly. Each catch adds 1 point and spawns a new butterfly.

Use **Restart** to begin a fresh round.

## Test

Run the core game-logic tests:

```bash
npm test
```

The current tests cover state creation, butterfly click detection, dog movement, timer ending, and scoring.
